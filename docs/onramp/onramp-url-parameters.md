# Welcome to Onramp & Offramp

Coinbase's Onramp & Offramp APIs and SDKs enable developers to move money seamlessly between fiat and onchain economies.

**[Onramp API / SDK](/onramp-&-offramp/onramp-apis/onramp-overview)**: Users can fund their onchain apps from their fiat accounts and crypto balances on Coinbase.

* **[Guest Checkout](/onramp-&-offramp/onramp-apis/onramp-overview#from-a-debit-card-guest-checkout)**: Enables US users without a Coinbase account to deposit up to \$500 per week with Apple Pay and debit cards.

**[Offramp API / SDK](/onramp-&-offramp/offramp-apis/offramp-overview)**: Users can cash out money onchain into fiat currencies (eg. USD).

Interested in using Coinbase Onramp and Offramp? Apply for access [here](https://support.cdp.coinbase.com/onramp-onboarding).
